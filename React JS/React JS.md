### Create a New Project

- ```npm create vite@latest my-vite-app```
- ```cd my-vite-app```
- ```npm install```
- ```npm run dev```


### Install Bootstrap

- ```npm install react-bootstrap bootstrap```

   Import Bootstrap CSS:

   ``` JS
   import 'bootstrap/dist/css/bootstrap.min.css'; 
   ```
